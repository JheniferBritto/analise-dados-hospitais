# 📊 Análise de Dados Hospitalares com Python

Projeto prático utilizando Pandas para limpeza, tratamento e análise de dados.

## 🔧 Etapas do Projeto

- Limpeza de dados
- Tratamento de valores nulos
- Conversão de datas
- Padronização de informações
- Análise exploratória

## 📈 Análises realizadas

- Hospital com mais pacientes
- Cidade com maior custo
- Custo por paciente
- Identificação de outliers

## 🚀 Tecnologias usadas

- Python
- Pandas
- Jupyter Notebook
